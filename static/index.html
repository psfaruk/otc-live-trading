      What Plybit AI does

      Streams live OTC forex candles from Quotex and runs a set of price-action theories (tick pressure, wick rejection, candle patterns, multi-timeframe trend) against every closed candle to produce a CALL/PUT read with a STRONG/MEDIUM/WEAK strength label.

      • Only forex pairs at or above the live payout floor stream continuously — everything else is blocked from opening.
      • Each theory's real accuracy is tracked over a rolling 7-day window; any theory measured below coin-flip is automatically muted until it recovers.
      • Strength reflects how much independent evidence backs a signal, not a promised win rate.

      ⚠ Read this before trading

      This project measured the OTC feed directly — a 6,000-candle statistical test and a separate 42,000-signal live production record — and both agree: the feed behaves like an unbeatable RNG. No theory, combination, or strength tier has shown a real edge above 50%.

      This app exists to analyze that feed honestly, not to promise winning trades. Treat every signal as informational, never as a guarantee.
